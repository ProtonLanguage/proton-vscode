# Proton# VS Code Extension

Official VS Code language support for the Proton# scripting language.

## Features

- ✅ Syntax highlighting for `.pros` files
- ✅ Proton# Dark theme
- ✅ Autocomplete for keywords, built-ins, and datastore functions
- ✅ Code snippets for common patterns
- ✅ Case error detection (Proton is case sensitive!)
- ✅ Missing semicolon warnings
- ✅ Module vs local variable conflict detection
- ✅ Auto-closing brackets, quotes, and block comments
- ✅ Auto-indent inside `do...end` blocks

## Snippets

| Prefix | Description |
|--------|-------------|
| `var` | Local variable |
| `const` | Local constant |
| `module` | Global module variable |
| `func` | Function definition |
| `funcr` | Function with return |
| `if` | If statement |
| `ife` | If / else |
| `ifee` | If / else if / else |
| `while` | While loop |
| `for` | For loop |
| `foreach` | For each loop |
| `try` | Try / catch |
| `print` | Print statement |
| `dssave` | Datastore save |
| `dsload` | Datastore load |
| `server` | Server script template |
| `bcom` | Block comment |
| `@lua` | Lua backend hint |
| `@python` | Python backend hint |

## Theme

This extension includes the **Proton# Dark** theme. To activate it:
1. Press `Ctrl+Shift+P`
2. Type `Color Theme`
3. Select `Proton# Dark`
